## About

Laravelを使ったアンケートフォームシステムです。

- 主にQRコードをスキャンしてスマホで答えてもらうよう
- 聞き取りスタッフが直接やりとりしやすいようにもできている
- スタッフや管理者がリアルタイムで集計閲覧できるようになっている
